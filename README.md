# Personal Finance Tracker

A Streamlit-based Personal Finance Tracker web application that helps users manage their income, expenses, savings, and financial records efficiently.

## Features
- Add income and expense records
- Track spending habits
- Visualize financial data
- Simple and user-friendly interface
- Real-time calculations and summaries

## Technologies Used
- Python
- Streamlit
- Pandas
- Matplotlib
